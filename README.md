# Steel DashBoard - Manufacturing Operations Portal

A centralized, high-performance launcher for manufacturing and operations applications. This portal provides a unified interface for managing industrial workflows, from crane maintenance to inventory optimization.

## 🚀 Overview

The **Manufacturing Operations Portal** is designed to streamline access to various specialized modules used in steel production and yard management. It offers a clean, responsive dashboard with quick-access cards for all critical operational tools.

### Key Modules
- **Maintenance**: Crane Maintenance, Tooling Management.
- **Production**: Tube Mill Monitoring, CTL & CRS Optimizers.
- **Logistics**: Yard Management (YMS), Truck TAT Tracking, Weighbridge Integration.
- **Inventory**: RM Stock & Stores Inventory Management.
- **Sales/Planning**: Order Management System (OMS).

## 🛠️ Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Radix UI](https://www.radix-ui.com/) & [Lucide React Icons](https://lucide.dev/)
- **Deployment**: [Vercel](https://vercel.com/)
- **Analytics**: Vercel Analytics

## ⚙️ Getting Started

### Prerequisites
- Node.js 18.x or later
- pnpm (recommended) or npm/yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd Steel-DashBoard
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Run the development server:
   ```bash
   pnpm dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure

- `/app`: Next.js pages and layouts.
- `/components`: Reusable UI components (AppCard, Footer, Header).
- `/public`: Static assets (Logos, Icons, Favicon).
- `/lib`: Utility functions and shared logic.
